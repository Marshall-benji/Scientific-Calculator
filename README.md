# 🧮 Scientific Calculator (Python Tkinter)

A clean and user-friendly **scientific calculator desktop application** built using Python and Tkinter.
This project demonstrates GUI development, event handling, and implementation of mathematical operations.

---

## 🚀 Features

* 🌙 Dark / Light Theme Toggle
* 🧪 Scientific Functions

  * sin, cos, tan
  * log, square (x²), square root (√)
  * factorial (!)
* 🕘 History Panel (tracks all calculations)
* 🗑️ Clear History option
* ⌨️ Keyboard Input Support
* ❌ Error Handling for invalid inputs

---

## 🛠️ Tech Stack

* Python
* Tkinter (GUI Library)
* Math Module

---


## 🎯 Learning Outcomes

* GUI development using Tkinter
* Event-driven programming
* Handling user inputs and errors
* Building desktop applications
